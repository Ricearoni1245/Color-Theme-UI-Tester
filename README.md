# Color Theme UI Tester

## Purpose
This was created for the purpose of
- better understanding typescript types and react usestate
- a tool to see the color themes users plan out visualized on a mock UI.

---

## In Progress
Currently, this repo has:
- configured themes in `theme.ts` and applied appropriate types
- initialized usestate in `app.tsx`
- created a tailwind.css theme system in `app.css`
- Created preview UI showcasing color elements and showcase
- Navbar component
- React Routing

---

## Future Vision
- Users inputs:
  - Color Theme
  - Images
  - Font
- Showcases Color Scheme/Font/Images
- Formats into a mock UI (e-commerce, portfolio, and informational view)
